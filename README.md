This program takes the excel file from the Church excel file that contains the lyrics.  It cleans out all unneeded rows and columns and outputs a clean .json file.  
"Verse 1" adds "1: " to the first word of the lyrics and "Chorus" adds "Ch: ".  Also adds pipe separator between each word.

This is intended to add to a library of lyrics in .json.  The format looks like this:
